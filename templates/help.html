Pro Optimization Tips

1

Start Conservative

Begin with 0.9 keep ratio, then gradually decrease based on accuracy metrics. Monitor validation loss closely.

💡 Tip: Use higher keep ratios for smaller models.
2

Stability First

If retraining is unstable, reduce learning rate by 10x and increase epochs. Use gradient clipping.

⚙️ Recommended: LR = 1e-4, Epochs = 100 for most models.
3

Compare Religiously

Always check insights after each run. Look for accuracy drop vs compression gain sweet spot.

📊 Metric: Target < 2% accuracy drop for> 3x compression.
4

Layer-wise Analysis

Different layers have different sensitivity. Use the Architecture page to identify robust layers.

🔍 Attention layers in Transformers are more sensitive than FFN layers.
