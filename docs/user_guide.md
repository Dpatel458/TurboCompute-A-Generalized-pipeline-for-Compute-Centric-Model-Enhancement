# How to Use the Platform

## Step 1: Select Model Type

Choose from:
- FNN
- CNN
- Transfer learning models
- Transformer models

## Step 2: Upload Model

Upload model in supported format.

If FNN:
- Upload dataset as well.

## Step 3: Select KEEP Ratio

Choose sparsity level.

## Step 4: Apply Pruning

System generates pruned model.

## Step 5: Choose Next Action

- Download pruned model
- Compare models
- Retrain (optional)

## Step 6: Final Download

Download:
- Pruned model
- Retrained pruned model
