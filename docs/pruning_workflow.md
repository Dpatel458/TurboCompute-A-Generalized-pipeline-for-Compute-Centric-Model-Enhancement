# Pruning Workflow

After uploading the model and selecting KEEP ratio:

1. The system applies structured pruning.
2. A pruned version of the model is generated.

## Post-Pruning Options

Users can:

- Download the pruned model
- Compare base vs pruned model
- Proceed to optional retraining

## Model Comparison

Users can inspect:
- Parameter reduction
- Structural changes
- Sparsity level
- Model size difference
