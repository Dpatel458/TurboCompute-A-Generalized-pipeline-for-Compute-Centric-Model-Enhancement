# Mask & Prune – Smart Model Compression Platform

## Why This Project Exists

Modern deep learning models are powerful but often large, slow, and energy-intensive. Our platform focuses on reducing model size while maintaining performance.

We aim to:
- Improve efficiency
- Reduce computational cost
- Promote Green AI
- Make models lightweight and deployable

## What Our Platform Does

Our system allows users to upload their neural network models and apply structured pruning using a configurable KEEP ratio.

The output is:
- A pruned model
- Comparison with the base model
- Optional retrained pruned model

## Key Benefits

- Lightweight Models
- Reduced Memory Usage
- Faster Inference
- Energy Efficient (Green AI aligned)
- Controlled sparsity via KEEP ratio

## Performance Focus

Our approach preserves model accuracy while reducing redundant parameters. We balance sparsity and performance carefully.
